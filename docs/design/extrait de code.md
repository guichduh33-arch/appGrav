Voici les extraits de code structurés pour **Tailwind CSS** et **CSS pur**, optimisés pour être injectés directement dans votre projet **Antigravity**. Ces extraits permettront à vos agents Claude Code d'appliquer le design "The Breakery" avec une précision chirurgicale. ### 1. Configuration du Thème (Tailwind Config) Ajoutez ces extensions dans votre fichier `tailwind.config.js` pour centraliser les tokens de design. ```javascript module.exports = { theme: { extend: { colors: { breakery: { onyx: '#0D0D0F', // Fond principal card: '#161618', // Fond des cartes gold: '#C9A55C', // Couleur accent stone: '#E5E7EB', // Texte principal smoke: '#9CA3AF', // Texte secondaire muted: '#ffffff14', // Bordures et séparateurs } }, fontFamily: { sans: ['Inter', 'sans-serif'], serif: ['Cormorant Garamond', 'serif'], // Pour les titres luxe }, borderWidth: { 'luxe': '1px', } }, }, } ``` ### 2. Classes de Composants Core (CSS/Utility) À insérer dans votre fichier de styles globaux (`globals.css`). ```css @layer components { /* Layout de base */ .layout-container { @apply min-h-screen bg-breakery-onyx text-breakery-stone font-sans; } /* Cartes de Pâtisserie / Produits */ .product-card { @apply bg-breakery-card border border-breakery-muted rounded-sm transition-all duration-300; @apply hover:border-breakery-gold/50 hover:shadow-[0_0_20px_rgba(201,165,92,0.1)]; } /* Bouton Primaire 'Luxe' */ .btn-gold { @apply bg-breakery-gold text-breakery-onyx font-semibold px-6 py-2 rounded-sm uppercase tracking-wider text-sm; @apply hover:bg-opacity-90 active:scale-95 transition-all; } /* Entrées de formulaire (Inputs) */ .input-breakery { @apply bg-transparent border border-breakery-muted text-breakery-stone px-4 py-2 rounded-sm; @apply focus:outline-none focus:border-breakery-gold focus:ring-0 transition-colors; @apply placeholder:text-breakery-smoke/50; } /* Status Badges */ .badge-status { @apply px-2 py-0.5 rounded-full text-[10px] font-bold uppercase tracking-widest border; } .badge-success { @apply bg-green-500/10 text-green-500 border-green-500/20; } .badge-warning { @apply bg-breakery-gold/10 text-breakery-gold border-breakery-gold/20; } .badge-error { @apply bg-red-500/10 text-red-500 border-red-500/20; } } /* Alignement des chiffres financiers */ .tabular-data { font-feature-settings: "tnum" on, "lnum" on; } ``` ### 3. Exemple de Structure React/Tailwind (Sidebar & Dashboard) Voici comment Claude Code devrait assembler une partie du Back-office. ```jsx const Sidebar = () => ( <aside className="w-[260px] h-screen bg-breakery-onyx border-r border-breakery-muted flex flex-col p-6"> <div className="mb-12"> <h1 className="font-serif text-2xl text-breakery-gold italic">The Breakery</h1> </div> <nav className="space-y-4"> <NavItem icon={<DashboardIcon />} label="Dashboard" active /> <NavItem icon={<StockIcon />} label="Stock & Inventory" /> <NavItem icon={<ChefIcon />} label="Production" /> </nav> </aside> ); const MetricCard = ({ title, value, change }) => ( <div className="bg-breakery-card p-6 border border-breakery-muted rounded-sm"> <p className="text-xs uppercase tracking-widest text-breakery-smoke mb-2">{title}</p> <div className="flex items-baseline justify-between"> <h3 className="text-3xl font-light tabular-data">{value}</h3> <span className="text-green-500 text-xs tabular-data">+{change}%</span> </div> </div> ); ``` ### 4. Directives pour les Graphiques (Recharts / Chart.js) Pour les graphiques de flux de stock : - **Stroke color :** `#C9A55C` (Gold) - **Area Fill :** Gradient de `#C9A55C` à `transparent`. - **Grid Lines :** `#ffffff08`. - **Tooltips :** Fond `#161618` avec bordure `#C9A55C`.
